# Troubleshooting Guide

## Common Issues

### Account & Workspace Setup

#### Unable to Sign Up
- Verify your email address is correct
- Check if you're using a supported browser (Chrome, Firefox, Safari)
- Clear your browser cache and try again

#### Can't Access Workspace
- Ensure you're logged in with the correct account
- Check if you have the necessary permissions
- Contact your workspace administrator

### KIVA Platform

#### Google Search Console Connection
- Ensure you're using the correct Google account
- Verify you have the necessary permissions in GSC
- Try disconnecting and reconnecting the integration

#### Keyword Analysis Issues
- Verify your date range selection
- Check if your domain is correctly set up
- Ensure your GSC property is properly verified

#### Content Generation
- Make sure you've selected the required elements:
  * LSI Keywords
  * SERP Intents
  * User Intent
  * PAA Questions
- Try regenerating the outline if results aren't satisfactory
